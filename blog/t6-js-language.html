Javascript Language

27 January 2016

In the JavaScript language, describe all the use cases for each of:

Parentheses()
These can be found when following a function and are used to contain the functions arguement.
Brackets []
Brackets are using to reference and array, they signal the beginning and end of an array objects and anything found between them is considered and object of that array. The can also be used as a way to reference an object in an array and are commonly found in for loops.
Braces {}
Code found inside the braces are usually part of a wider statement and/or function and are used to contain the andy code statements, functions or other expressions.
Single quotes ''
Single quotes are usually used to contain a text string, or can be used when referencing the name of an object eg a a html ID or Class.
Double quotes ""
Double quotes can perform the same function as single quotes, and can be used if you need to do a quote inside a string which is already using single quotes.
