Udacity Reflection - Part 2

What is grid based design?

Grid based design, is the design of a webpage based on ordering content based on an invisible grid that is across your webpage. This grid design is used in conjunction with the users natural eye flow of reading from left to right and top to bottom. Thinking of the page as a gird, when combined with understanding how a user is likely to look at a page helps designers and developers to organise content, and draw users to a certain section or sections of a page.

What’s all the hype about responsive webpages?

Responsive web design is vital given the wide range of devices users are using to browse the web. In the past developers’ would create a site specifically for desktop and another specifically for mobile - essentially they have built two separate sites. However, responsive sites allow developers to create one site that will dynamically adjust itself to provide the user the best experience regardless of what device they are using to view the page.

What is semantic structure?

Semantic structure is the use of html tags that make sense to both the web browser and to and people who are reading them. They are tags such as <h1> for heading, <nav> for navigation or <img> for image.

Any other points you found interesting and want to reflect on.
